Privacy and Security Policy

Last Updated:04/08/2025

This privacy policy describes how the E-commerce app collects, uses, and protects user information.

1. Information Collection

The E-commerce app collects only the necessary information to authenticate users. This may include:

  • Email address
  • Password
  • Full name

2. Use of Information

The information collected is used solely for the following purposes:

  • User authentication and secure account access.
  • Improving the user experience within the app.

3. Sharing of Information

We do not share, sell, or transfer user personal information to any third-party companies or organizations.

4. Data Deletion

If a user chooses to delete their account, all associated data (including email, name, and authentication data) will be permanently deleted from our servers.

5. Security

We take reasonable technical and organizational measures to protect personal data against unauthorized access, loss, or destruction. We use secure platforms such as Firebase and Supabase that provide strong security protocols and encryption.

6. Changes to This Policy

We reserve the right to update this privacy policy at any time. Any changes will be posted on this page with an updated date.

7. Contact
